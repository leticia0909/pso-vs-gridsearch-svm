# pso-vs-gridsearch-svm
Este projeto compara a eficácia e eficiência de dois métodos distintos para a otimização de hiperparâmetros ($C$ e $\gamma$) de um Support Vector Machine (SVM) com kernel RBF: Grid Search (GS) e Particle Swarm Optimization (PSO). Objetivo é maximizar o desempenho do classificador na tarefa de classificação do dataset Breast Cancer Wisconsin.
